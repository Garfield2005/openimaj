44  /**
45   * A hash function factory that produces hash functions that approximate L1
46   * (city-block) distance in closed spaces using random axis-aligned hyperplanes.
47   * <p>
48   * The hash function hashes the input vector into a binary value (i.e. 0 or 1). 
49   * It works by choosing a random dimension and a random threshold along that 
50   * dimension (between a given minimum and maximum which define the closed space).
51   * Input vectors whose element at the chosen dimension is greater than or equal
52   * to the threshold generate a 1; values less than the threshold generate a 0. 
53   * 
54   * @author Jonathon Hare (jsh2@ecs.soton.ac.uk)
55   */
